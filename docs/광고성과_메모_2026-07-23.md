## 💡 액션
1. CTR 0.6% 이하(소재 2,4,5) OFF → 예산 소재 1,6에 집중
2. 신규 소재는 승자 공식 복제
3. ⚠️ 다음 확인 필수: 클릭 46 → 실제 진단완료 DB 몇 건? (Supabase diagnoses)
   - 클릭만 많고 DB 0이면 소재가 아니라 랜딩(진단화면) 이탈 문제
